Recipe Transformer

Teammates: Youngeon Kim, Kevin Mui, Ignacio de Osma
Programming language used: Python 3
Modules (you can download these using pip): nltk, urllib.request, BeautifulSoup, difflib, random, copy, re, sys

The file can be run by the following command: python transformations.py. You will be prompted to enter a url and choose a transformation (by number).
